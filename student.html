Round 4 • Debugging

Bug Detective

This time the code is broken. Your job is to fix it.

Mission
  • Fix the if statement syntax.
  • Run the code and click the button.
  • Win by making the page show the success message.
Coach tip
Debugging is normal. The goal is not “never break code.” The goal is “notice what is broken and fix one part at a time.”
